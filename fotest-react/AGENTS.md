# fotest-react Agent Metadata

## Purpose
Frontend React app for managing persons via FoTestApi backend.
- CRUD UI with search and forms
- API integration using Axios
- Uses Material UI with dark theme

## Agent role
- Build and maintain UI components
- Ensure API endpoints are called correctly
- Handle state, error display, and form validation

## Commands
- `npm install`
- `npm start`
- `npm run build`
