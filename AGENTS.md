# AGENTS Index

This project currently uses one main assistant workflow (the public GitHub Copilot + VS Code Copilot agent setup), but here we define an explicit structure for future agent modules.

## Agents
- `agents/main-agent.md`: Primary development assistant for coding, refactoring, and testing.
- `agents/release-agent.md`: Optional future agent for release and deployment automation.
- `agents/docs-agent.md`: Optional future agent for docs, API reference, and README updates.

## Usage
Open the appropriate agent file in `agents/` for role details and instructions.
